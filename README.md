### Core Projects

#### [Spectra](https://github.com/codex-mohan/spectra) *(Flagship)*
A fast and minimal AI agent library and SDK built for TypeScript and Rust environments.

#### [NovaFlow](https://github.com/codex-mohan/novaflow)
An intelligent desktop voice assistant featuring image/voice recognition and a modular, node-based user interface. Built with Tauri, Next.js, and Rust, running on Bun.

#### [Horizon](https://github.com/codex-mohan)
An agentic AI assistant engineered using LangGraph, Next.js, and Docker to execute system-level operations and handle complex LLM orchestration states.
